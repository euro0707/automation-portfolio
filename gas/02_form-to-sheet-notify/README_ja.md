# Googleフォーム → スプレッドシート → メール通知 自動化

お問い合わせフォームの回答を自動でシートに記録し、管理者と回答者の両方にメールを送信します。外部サービス不要。

---

## 処理の流れ

1. 訪問者がGoogleフォームに回答を送信
2. 回答がスプレッドシートに自動記録
3. 管理者にメール通知（問い合わせ内容＋シートへのリンク）
4. 回答者に自動返信メール（受付確認）
5. スプレッドシートの該当行に「通知済み」と記録

## セットアップ手順

### 1. Googleフォームを作成する

以下のフィールドを追加（名前は任意に変更可）:

| フィールド | 種類 |
|-----------|------|
| お名前 | 記述式（短文） |
| メールアドレス | 記述式（短文） |
| お問い合わせ種別 | プルダウン |
| お問い合わせ内容 | 記述式（長文） |
| 緊急度 | 選択式 |

### 2. フォームをスプレッドシートに連携する

フォーム編集画面 → **回答** タブ → スプレッドシートアイコン → 新しいスプレッドシートを作成

### 3. スクリプトを設定する

`form_notify.js` を開いてIDを設定:

```js
var SPREADSHEET_ID = "スプレッドシートのID";
```

スプレッドシートIDはURLの以下の部分:  
`https://docs.google.com/spreadsheets/d/【ここ】/edit`

### 4. トリガーを登録する（初回のみ）

GASエディタで `setupTrigger` を選択 → 実行  
権限の許可ダイアログが表示されたら承認する

### 5. 動作確認

GASエディタで `testNotification()` を実行 → メールが2通届けば成功

## 主要な関数

| 関数名 | 処理内容 |
|--------|---------|
| `onFormSubmit(e)` | フォーム送信時に自動実行 |
| `sendAdminNotification()` | 管理者への通知メール送信 |
| `sendAutoReply()` | 回答者への自動返信メール送信 |
| `setupTrigger()` | トリガー登録（初回のみ実行） |
| `testNotification()` | 手動テスト（実際の送信なし） |
